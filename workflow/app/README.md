# App

This folder contains the user-facing part of the project. The Streamlit app is the main entry point for exploring Pokémon team optimization results.

## Key Files
- **streamlit_app.py**: interactive UI for team generation, analysis, and result browsing.

## What It Shows
- generated teams and their fitness values
- team archetypes and role balance
- analysis views for strengths, weaknesses, and coverage

## Design Goal
The app is intentionally lightweight: the interface is for presenting the project clearly, while the optimization work stays in the core workflow code under `src/`.

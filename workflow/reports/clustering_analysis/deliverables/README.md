# Clustering Deliverables

This folder packages the clustering work into reusable outputs: trained models, summary reports, and visual artifacts.

## What It Contains
- persisted clustering models for inference
- archetype summary tables and example mappings
- comparison notes that justify the chosen clustering approach
- plots that visualize how Pokémon group into roles

## Why It Matters
The clustering work is not just a side experiment. It feeds the team optimizer by giving each Pokémon a role label that can be used in fitness scoring and team balance analysis.
